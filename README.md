## About

This server is a gateway for the [Chucknorris API](https://api.chucknorris.io)
it is used alongside with the desafio-fullstack-chucknorris-facts frontend.

## Available Commands

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```
